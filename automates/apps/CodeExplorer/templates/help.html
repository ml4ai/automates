About
Input Fortran code in the code editor area and hit the 'Submit' button. Three examples are provided to get started. The first one is a toy program that demonstrates basic Fortran code idioms (main, subroutine, loops, conditionals). The second and third programs are implementations of two different models of calculating potential evapotranspiration, taken from DSSAT.
  • PETPT: The Pristley-Taylor model of evapotranspiration.
  • PETASCE: The ASCE standard evapotranspiration calculation method.
Things you can try doing:
  • Click on the square nodes to view the equation that is automatically constructed from the code, and typeset with LaTeX., as well as the Python 'lambda' function that is generated.
    • L: Literal value assigment (i.e. no inputs to the function)
    • A: Assign nodes
    • C: Condition nodes
    • D: Decision nodes
  • Click on the round nodes to view metadata about the variable that has been extracted using NLP techniques from the comments in the source code as well as scientific papers about these models (note that the NLP results shown in this demo are precomputed rather than running live.)
  • Rename a variable in the code and resubmit it for processing (make sure to rename all instances of it in the code!)
